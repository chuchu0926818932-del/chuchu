# SNL 短影音企劃工作台

這是一個本機優先的短影音拍攝工作台，整合 80 條題目、可直接拍攝的完整腳本、8 種爆款公式與 SNL 的品牌安全原則。未登入時保存在本機；登入後會同步到 Supabase，換電腦也能接著做。

正式網站：[https://snl-short-video-studio.vercel.app](https://snl-short-video-studio.vercel.app)

## 平常怎麼開啟

1. 雙擊桌面的「啟動短影音企劃網站」捷徑。
2. 等待瀏覽器自動開啟 `http://localhost:3000`。
3. 使用期間請保留啟動視窗；關閉啟動視窗即可停止網站。

也可以直接雙擊本資料夾內的 `啟動短影音企劃網站.bat`。

## 主要功能

- 待選題庫：搜尋、依公式／主題／風險篩選、收藏題目，預設排除已完成內容。
- 完整拍攝腳本：每支題目直接提供分秒口播、拍攝畫面與畫面字幕，可在預覽區修改並自動保存。
- 拍攝完成歸檔：按下完成後保存企劃紀錄，並從後續選題結果自動排除。
- 去重續題：首批 80 條完成後解鎖，依公式、領域、情境、受眾與目的每批產生 10 條，並比對標題與鉤子去重。
- 完成紀錄：追蹤待規劃、撰稿、拍攝、後製與完成時間，隨時回看完成腳本。
- 本機備份：可匯出／匯入 JSON 備份檔。
- Supabase 同步：Email 免密碼登入、跨裝置同步，本機離線備援。

## 資料保存

收藏、企劃、完整腳本、完成狀態與去重新題目都會先保存在瀏覽器的本機儲存空間。登入 Supabase 後，資料也會同步到 `snl_workspaces`；資料表已啟用 Row Level Security，每位登入者只能讀寫自己的工作區。如果要清除瀏覽器資料，仍建議先在網站右上角按「匯出備份」。

## Supabase 設定

本機連線值放在不會提交到 GitHub 的 `.env.local`。範例格式位於 `.env.example`。

資料庫結構與 RLS 規則位於 `supabase/migrations/202607140001_create_snl_workspaces.sql`。專案的 Auth Site URL 需設為 `http://localhost:3000`，並啟用 Email 登入。

## 題庫更新

題庫來源位於桌面：`短影音文案\源源不絕主題庫`。

更新分類檔後，可執行 `scripts\import-topics.mjs` 重新匯入題庫，再重新啟動網站。
