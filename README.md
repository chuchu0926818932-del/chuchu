# SNL 短影音企劃工作台

這是一個只在本機運作的內容企劃網站，整合 80 條短影音題目、8 種爆款公式與 SNL 的品牌安全原則。

## 平常怎麼開啟

1. 雙擊桌面的「啟動短影音企劃網站」捷徑。
2. 等待瀏覽器自動開啟 `http://localhost:3000`。
3. 使用期間請保留啟動視窗；關閉啟動視窗即可停止網站。

也可以直接雙擊本資料夾內的 `啟動短影音企劃網站.bat`。

## 主要功能

- 靈感題庫：搜尋、依公式／主題／風險篩選、收藏題目。
- 企劃工作區：填寫受眾、目的、鉤子、核心觀點、鏡頭、CTA 與發布日。
- 複製給 Codex：把企劃整理成可直接貼到對話的製作指令。
- 主題生成器：組合公式、主題、情境、受眾與內容目的，產生下一批題目的提示詞。
- 製作看板：追蹤待規劃、撰稿、拍攝、後製與完成進度。
- 本機備份：可匯出／匯入 JSON 備份檔。

## 資料保存

收藏與企劃保存在瀏覽器的本機儲存空間，不會自動上傳。如果要清除瀏覽器資料或換電腦，請先在網站右上角按「匯出備份」。

## 題庫更新

題庫來源位於桌面：`短影音文案\源源不絕主題庫`。

更新分類檔後，可執行 `scripts\import-topics.mjs` 重新匯入題庫，再重新啟動網站。
